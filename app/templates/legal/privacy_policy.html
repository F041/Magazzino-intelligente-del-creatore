Norme sulla Privacy

Ultimo aggiornamento: 24 Settembre 2025

Dati raccolti

Questa applicazione, "Magazzino del Creatore", accede ai dati del tuo canale YouTube tramite le API di Google solo dopo la tua esplicita autorizzazione (protocollo OAuth2).

I dati a cui accediamo includono:

  • Metadati dei video (titolo, descrizione, data di pubblicazione).
  • Trascrizioni (sottotitoli) dei video, sia manuali che automatiche.

Salviamo queste informazioni nel database dell'applicazione per permetterti di effettuare ricerche sui tuoi stessi contenuti.

Come usiamo i tuoi dati

I dati recuperati dal tuo canale YouTube sono usati esclusivamente per le seguenti finalità:

  • Indicizzare i contenuti per creare una base di conoscenza interrogabile.
  • Fornire risposte alle tue domande basate sui contenuti che hai fornito.

I tuoi dati non vengono condivisi con terze parti, venduti o usati per scopi pubblicitari.

Servizi API di Google

L'uso delle informazioni ricevute dalle API di Google da parte di questa applicazione aderirà alle Norme sui dati utente dei servizi API di Google, inclusi i requisiti di utilizzo limitato.

Contatti

Per qualsiasi domanda riguardo questa privacy policy, puoi contattare lo sviluppatore.
